Julia wrapper for NVIDIA's cuDNN  library.

## Supported features

 * convolutions
 * pooling
 * batch normalization
 * softmax (4D)

See [tests](https://github.com/dfdx/CUDNN2.jl/blob/master/test/runtests.jl) for examples of usage.
